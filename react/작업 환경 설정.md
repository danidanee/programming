<h1>작업 환경 설정</h1>

<br/>

* Node.js, npm 설치

```
https://nodejs.org/ko/download/
window installer
```

<br/>

* 에디터 생성

```
https://code.visualstudio.com/download
```

<br/>

* 파일 생성

```
npm create react-app [frontend]
yarn create react-app [frontend]
```

<br/>

* 타입스크립트 파일 생성

```
npm create-react-app [frontend] --typescript
```

<br/>

* 파일 열기 및 실행

```
code [frontend]
npm install
npm start
```
